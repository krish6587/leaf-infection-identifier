# Automatic Leaf Infection Identifier

A machine learning-based application that detects and classifies plant leaf diseases from images. The system processes leaf images, extracts important features, and predicts whether the leaf is healthy or infected.

## Features
- Leaf image analysis and disease detection
- Machine learning-based classification
- User-friendly graphical interface
- Supports healthy and infected leaf identification

## Technologies Used
- Python
- OpenCV
- Machine Learning
- Image Processing

## Project Structure
- `main.py` – Main application entry point
- `classifier.py` – Disease classification logic
- `design.py` / `design.ui` – GUI implementation
- `requirements.txt` – Project dependencies
- `Image Dataset/` – Training and testing images
